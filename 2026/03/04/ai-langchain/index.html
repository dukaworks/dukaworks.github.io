LangChain 框架入门

发表于 2026-03-04 分类于 AI智能体

LangChain 是一个开发语言模型应用的框架，它提供了丰富的组件，简化了 AI 应用的开发过程。

LangChain 的核心概念、Model（语言模型）、Prompt（提示词）、Chain（链）、Agent（智能体）、Memory（记忆）。Model 负责生成文本，Prompt 封装提示模板，Chain 连接多个组件，Agent 决定行动，Memory 保存对话历史。

PromptTemplate 提供了灵活的提示词管理。可以定义变量，从输入值填充。支持 Few-shot 示例、输出解析器等高级功能。

Chain 是最常用的概念。SimpleSequentialChain 按顺序执行多个链，输出上一个链的输入下一个链。RouterChain 根据输入路由到不同的子链。

Agent 能够自主决定行动。ReAct Agent 交替思考和行动，Zero-shot Agent 一次性决定所有行动。Tool 是 Agent 可以使用的工具，如搜索、计算、API 调用等。

Memory 让模型记住对话历史。ConversationBufferMemory 保存所有历史，ConversationSummaryMemory 总结历史，节省 Token。

VectorStore 用于文档检索。LangChain 支持多种向量数据库、Chroma、Pinecone、FAISS 等。配合 Embeddings 模型，实现语义搜索。

LangChain 支持多种模型、OpenAI、Hugging Face、Anthropic、本地模型（如 Llama）。可以通过统一的接口切换不同模型。

使用 LangChain，开发者不需要关注底层细节，专注于业务逻辑。它大大降低了开发 AI 应用的门槛，让更多人能够构建智能应用。
