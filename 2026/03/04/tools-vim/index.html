Vim 编辑器使用

发表于 2026-03-04 分类于 实用工具

Vim 是强大的终端编辑器，以其高效的键盘操作和丰富的插件系统著称。虽然学习曲线陡峭，但掌握后能极大提升编辑效率。

Vim 有多种模式、普通模式（浏览、删除）、插入模式（输入文本）、可视模式（选择文本）、命令模式（执行命令）。ESC 键返回普通模式。

基本移动、h（左）、j（下）、k（上）、l（右）。w/W（下一个单词），b/B（上一个单词），e/E（单词末尾），0（行首），$（行尾），G（文件末尾）。

编辑操作、i（在光标前插入），a（在光标后插入），o（新起一行），dd（删除行），yy（复制行），p（粘贴），u（撤销），Ctrl+r（重做）。

搜索替换、/pattern（向下搜索），?pattern（向上搜索），n/N（下一个/上一个），:s/old/new/g（全局替换当前行），:%s/old/new/g（全局替换文件）。

分屏操作、:split（水平分屏），:vsplit（垂直分屏），Ctrl+w+w（切换窗口），Ctrl+w+q（关闭窗口）。

Vim 配置文件、~/.vimrc。常用配置、行号 set number，语法高亮 syntax on，缩进设置 set tabstop=4 shiftwidth=4。

Vim 插件管理器、Vundle、vim-plug、dein.vim。常用插件、NERDTree（文件树）、CtrlP（模糊搜索）、vim-airline（状态栏）。

Vimscript 是 Vim 的脚本语言，可以自定义函数和命令。对于复杂的配置，学习 Vimscript 很有帮助。

Vim 是 Linux 服务器编辑文件的利器。即使日常使用 VS Code，掌握 Vim 也能在 SSH 远程编辑时游刃有余。
