TypeScript 类型系统

发表于 2026-03-04 分类于 Web开发

TypeScript 是 JavaScript 的超集，添加了静态类型检查。它提高了代码的可维护性和开发体验，是现代前端开发的标准选择。

基本类型、string、number、boolean、null、undefined、any、unknown、never、void。any 表示任意类型，unknown 是更安全的 any，never 表示不可能存在的类型。

类型注解、let name: string = "hello"。TypeScript 可以自动推断类型，大部分时候不需要显式注解。

数组类型、let nums: number[] = [1、 2、 3] 或 let nums: Array<number> = [1、 2、 3]。元组类型、let pair: [string、 number] = ["hello"、 42]。

对象类型、let user: { name: string; age: number } = { name: "John"、 age: 30 }。可选属性用 ?、{ name: string; age?: number }。

函数类型、function greet(name: string): void { console.log(name); }。箭头函数、const add = (a: number、 b: number): number => a + b。

泛型、function identity<T>(arg: T): T { return arg; }。泛型让函数和类可以处理多种类型。

高级类型、联合类型 string | number，交叉类型 A & B，字面量类型 "a" | "b"，类型断言 value as string。

类型守卫和类型谓词、typeof 和 instanceof 检查类型，isString(x: unknown): x is string 定义谓词。

类型推断是 TypeScript 的强大功能。大多数时候不需要手动标注，让编译器推断。只在必要时明确类型。

TypeScript 的类型系统既有深度又有广度。掌握它需要时间，但回报巨大。大型项目用 TypeScript 能避免很多错误。
