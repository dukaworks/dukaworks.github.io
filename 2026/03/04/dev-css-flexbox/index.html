Flexbox 布局完全指南

发表于 2026-03-04 分类于 Web开发

Flexbox（Flexible Box）是 CSS3 引入的布局模式，专门用于一维布局。它让复杂的布局变得简单直观。

Flexbox 的核心概念、容器和项目。设置 display: flex 的元素是容器，它的直接子元素是项目。容器有主轴和交叉轴，默认主轴是水平方向。

容器属性、flex-direction 决定主轴方向（row、column），justify-content 决定主轴对齐（flex-start、center、space-between），align-items 决定交叉轴对齐，flex-wrap 决定是否换行。

项目属性、flex-grow 决定放大比例，flex-shrink 决定缩小比例，flex-basis 决定初始大小。order 决定显示顺序，align-self 覆盖对齐方式。

Flexbox 解决了很多经典布局问题、垂直居中只需要 justify-content: center; align-items: center;，等高列自动实现，响应式布局用 flex-wrap 配合百分比宽度。

Flexbox 适合一维布局，如导航栏、卡片列表。二维布局应该使用 Grid。Grid 和 Flexbox 可以结合使用，发挥各自优势。

Flexbox 的浏览器支持已经很完善，但在一些老旧浏览器中可能需要前缀。使用 Autoprefixer 工具可以自动添加兼容性前缀。

掌握 Flexbox 是现代前端开发的必备技能。它简化了布局代码，提高了开发效率，值得深入学习。
