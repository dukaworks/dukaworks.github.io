CSS Grid 布局详解

发表于 2026-03-04 分类于 Web开发

CSS Grid 是最强大的 CSS 布局系统，专门用于二维布局。它可以同时处理行和列，实现复杂的网格布局。

Grid 的核心概念、容器、轨道、单元格、区域。设置 display: grid 的元素是容器，轨道是行或列，单元格是行和列的交叉点，区域是任意单元格组合。

定义网格、grid-template-columns 定义列轨道，grid-template-rows 定义行轨道。使用 repeat() 函数简化重复定义，fr 单位表示比例。例如、grid-template-columns: repeat(3、 1fr) 定义三个等宽列。

定位项目、grid-column 和 grid-row 定位项目，可以跨越多列多行。使用命名网格线更直观、grid-template-columns: [start] 1fr [mid] 2fr [end]。

区域布局、grid-template-areas 定义区域，grid-area 将项目放入区域。这种方式代码语义化，易于理解和维护。

Grid 布局技巧、隐式网格自动创建行列，grid-auto-flow 控制填充顺序，minmax() 设置最小最大尺寸，auto-fit 和 auto-fill 实现响应式。

Grid 的对齐功能丰富、justify-items、align-items 控制单元格对齐，justify-content、align-content 控制整体对齐。

Grid 适合二维布局，如页面整体布局、仪表盘、图片画廊。与 Flexbox 配合使用，可以实现任意复杂的布局。

Grid 的浏览器支持已经很好，现代浏览器全面支持。使用 Autoprefixer 处理兼容性，放心使用。
