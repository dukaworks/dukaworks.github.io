Markdown 写作技巧

发表于 2026-03-04 分类于 实用工具

Markdown 是一种轻量级标记语言，用简洁的语法实现富文本格式。掌握它能让写作更高效。

基础语法包括、# 标题、**粗体**、*斜体*、~~删除线~~、[链接](url)、![图片](url)、> 引用、- 列表。这些简单符号就能表达大部分文本格式。

代码块使用三个反引号包裹，可以指定语言实现语法高亮、javascript ... 。行内代码用单个反引号、code。

表格使用 | 分隔列，- 分隔表头和内容。对齐方式可以通过在 --- 前加冒号实现、---: 右对齐，:---: 居中。

任务列表用 - [ ] 和 - [x] 表示未完成和已完成。这对写待办事项很有用。

高级特性包括、脚注、定义列表、数学公式（需要支持）、图表等。不过不同编辑器支持程度不同。

推荐使用 Typora、VS Code、Obsidian 等 Markdown 编辑器。它们提供实时预览、快捷操作、插件扩展等功能。

Markdown 的简单易用使其成为技术写作的首选。花点时间学习，你会发现写作变得如此轻松。
