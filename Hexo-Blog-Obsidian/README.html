Obsidian Hexo 写作仓库

这是 Hexo 博客的 Obsidian 写作仓库，用于管理博客文章。

目录结构

_obsidian/
├── _templates/          # 文章模板
│   ├── hexo-post.md    # 发布文章模板
│   └── hexo-draft.md   # 草稿模板
├── _posts/             # 发布的文章（同步到 Hexo）
├── _drafts/            # 草稿（不发布）
└── images/             # 本地图片（可选）

使用步骤

1. 在 Obsidian 中打开此文件夹

  • 打开 Obsidian
  • 选择 “Open folder as vault”
  • 选择 source/_obsidian 目录

2. 安装必要插件

  • Templater - 模板功能
  • Paste image rename - 图片重命名

3. 配置 Templater

设置 → Templater：

  • Template folder location: _templates/
  • Trigger Templater on new file creation: 开启

4. 创建文章

方式1：使用模板

  • Ctrl+P → Templater: Open Insert Template Modal
  • 选择 hexo-post 或 hexo-draft

方式2：手动创建

  • 在 _posts/ 或 _drafts/ 目录创建 Markdown 文件
  • 复制模板内容

5. 写作

  • 正常写作
  • 使用 PicGo 上传图片到图床
  • 粘贴图片链接

6. 同步到 Hexo

完成文章后，复制到 Hexo：

# 从 _posts 复制到 Hexo
xcopy /Y "_obsidian\_posts\*.md" "..\_posts\"

# 或者手动复制粘贴

7. 部署

cd ../..
hexo clean && hexo g && hexo d

注意事项

  1. 文件名规范：使用英文或拼音，不要有特殊字符
  2. Front-matter：必须包含 title 和 date
  3. 图片：建议使用图床，不要放本地
  4. 分类标签：在 front-matter 中设置

模板变量

变量 说明 示例
{{title}} 文章标题 我的文章
{{date:YYYY-MM-DD HH:mm:ss}} 当前日期时间 2026-03-06 14:30:00

快捷操作

操作 命令
插入模板 Ctrl+P → Templater
上传图片 PicGo 快捷键
预览模式 Ctrl+E
